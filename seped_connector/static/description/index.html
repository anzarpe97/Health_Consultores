SEPED Connector

Integración API SEPED — Inventario y Clientes

Módulo de integración unidireccional Odoo → SEPED. Permite sincronizar el catálogo de productos, actualizar cantidades de stock y enviar el padrón de clientes al sistema SEPED de forma manual o automática.

Endpoints soportados

Full Sync Productos

POST /api/inventario/productos/sync — Envía el catálogo completo de productos activos.

Actualizar Stock

PATCH /api/inventario/productos/stock — Actualiza cantidades sin reenviar el catálogo.

Sync Clientes

POST /api/inventario/clientes/sync — Envía el padrón de clientes al distribuidor.

Características

  • Autenticación por header X-API-KEY
  • Sincronización en lotes configurable (batch_size)
  • Manejo de errores HTTP con mensajes descriptivos (401, 422, timeouts)
  • Wizard de sincronización manual desde el menú SEPED
  • 3 Cron jobs preconfigurados (activables desde Ajustes > Técnico > Automatización)
  • Registro de última ejecución por tipo de sync
